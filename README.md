# Retrieval Augmented Generation (RAG) System

## Overview

This project implements an end-to-end Retrieval Augmented Generation (RAG) pipeline that enhances Large Language Model (LLM) responses by retrieving relevant information from external documents before generating answers.

The system combines document processing, embedding generation, vector similarity search, and LLM-based response generation to provide accurate and context-aware answers.

---

## Features

- Document ingestion and preprocessing
- Text chunking for efficient retrieval
- Semantic search using embeddings
- FAISS vector database for similarity search
- LLM-based response generation
- Retrieval evaluation using Hit Rate, Recall, and MRR metrics
- Modular and maintainable project structure
